# Zizoo Front-End Code Challenge

## Goal

Create and showcase a responsive react component.

## Tasks

- Create a configurable responsive react component based on the this [zepelin.io scene](https://scene.zeplin.io/project/5aa2574b349af0b2682b1730).
- Create a trivial page showcasing an example result set as similar as possible to https://www.zizoo.com/en/search/. The showcase should be able to demonstrate both desktop and mobile views.
